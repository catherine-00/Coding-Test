# 3.1 제목과 문단
## `<h_>` : 제목 표시하기
- 제목은 글의 정체성을 드러내는 중요한 요소
- 제목 태그를 쓰면 일반적인 택스트보다 두껍게 표시된다.
- `<h` 뒤에는 1 ~ 6의 숫자 중 하나를 입력
    + 1 ~ 6은 제목 텍스트의 상대적인 크기를 나타내는 것
    + 크기 : 1 > ... > 6 
- 사용 예시
```html
<!DOCTYPE html>
<html lang="ko">
    <head>
        <meta charset="utf-8">
        <title> 제목 태그별 크기 비교</title>
    </head>
    <body>
        <h1>숫자가 1일 때의 제목 텍스트</h1>
        <h2>숫자가 2일 때의 제목 텍스트</h2>
        <h3>숫자가 3일 때의 제목 텍스트</h3>
        <h4>숫자가 4일 때의 제목 텍스트</h4>
        <h5>숫자가 5일 때의 제목 텍스트</h5>
        <h6>숫자가 6일 때의 제목 텍스트</h6>
    </body>
</html>
```

## `<p>` : 문단 표시하기
- 문단(단락) : 글을 내용상 끊어서 구분한 하나하나의 토막
- `<p>` 태그는 하나의 문단을 표시
- 다른 요소와의 간격을 위해 일정한 여백을 가짐
- 글의 본문을 표시할 때 주로 사용
```html
<!DOCTYPE html>
<html lang="ko">
    <head>
        <meta charset="utf-8">
        <title>문서 제목</title>
    </head>
    <body>
        <p>p 태그는 문단을 나타낼 때 사용하는 태그입니다.</p>
        <p>하나의 문단에는 일정한 여백이 형성되어 화면 상에서 구분하기가 용이합니다.</p>
    </body>
</html>
```

## `<blockquote>` : 인용구 표시하기
- 특별한 스타일이 가미된 문단을 표시하는 태그
- 일반적인 텍스트보다 안쪽으로 들여 써진다
- 선택적으로 `cite`라는 속성을 추가할 수 있음
    + 인용구의 출처를 입력하는 속성
    + 문구를 인용해 온 웹페이지의 주소를 입력함

```html
<body>
  <p>p 태그는 문단을 나타낼 때 사용하는 태그입니다.</p>
  <blockquote cite="https://wikidocs.net/book/7596">웹 개발 입문자들을 대상으로 하는 HTML 교재입니다.</blockquote>
</body>
```

## `<hr>` : 수평선 표시하기
- 단일 태그이다.
    + 닫는 태그가 필요 x
- 텍스트 문장 사이에 삽입되어 텍스트의 내용을 구분짓는 역할
```html
<body>
  <p>지금까지 스포츠 뉴스였습니다.</p>
  <hr>
  <p>날씨 예보입니다. 오늘은 하루 종일 맑았지만, 내일은 비가 올 예정입니다.</p>
</body>
```
- 기본 : 얇고 긴 검정색 선의 형태
    +  `color` 속성이나 `size` 속성으로 색깔과 두께를 설정할 수 있다.
```html
<body>
  <p>지금까지 스포츠 뉴스였습니다.</p>
  <hr color="purple" size="10">
  <p>날씨 예보입니다. 오늘은 하루 종일 맑았지만, 내일은 비가 올 예정입니다.</p>
</body>
```

# 3.1 HTML 코드의 특징
## HTML 코드가 해석될 때
- 엔터를 이용한 줄바꿈은 공백으로 처리됨
- 스페이스를 이용한 공백이 연달아 작성된 경우 한 번으로 인정

## HTML 코드와 개행 `<br>`
- 엔터를 이용한 줄바꿈은 웹 화면에서 텍스트 콘텐츠에 영향을 주지 않는다.
- 여러 번 엔터를 쳐도 해당 위치에 공백 하나만이 추가된다.
```html
<body>
  <p>구워먹고 볶아먹는 참 좋은 표고버섯, 나오셔서 구경하세요.</p>
  <p>구워먹고 볶아먹는 참 좋은 표고




    버섯, 나오셔서 

    구경하세요.</p>
</body>
```
- 줄바꿈은 `<br>` 태그를 사용한다.
    + 단일 태그이다
    + 원하는 만큼 줄바꿈을 표시할 수 있다.

```html
<body>
  <p>
    비가 오나 <br>
    눈이 오나 <br>
    바람이 부나
  </p>
  <p>
    비가 오나
    눈이 오나 
    바람이 부나
  </p>
</body>
```

## HTML 코드와 공백 `&nbsp;`
- 연달아 입력된 스페이스 키는 1번으로 인정된다.
- 연속적인 공백을 표시하고자 할 때는 `엔터티 코드`를 사용해야 한다.
- 엔터티 코드
    + `&` 기호로 시작해서 `;` 기호로 끝나는 문자의 조합
    + HTML 코드 내에서 특수 문자를 나타내기 위해 사용하는 약속
    + 그 중 공백은 `&nbsp;` 인 것.

## `<pre>` : 있는 그대로 표시하기
- 태그 안에 작성된 텍스트를 HTML 코드에 작성된 그대로 화면에 출력하는 태그
    + 개행이나 공백도 그대로 출력됨
- 웹 브라우저 화면에 소스 코드를 표현할 때 유용
```html
<body>
  <h2>이름을 출력하는 파이썬 코드</h2>
  <pre>


    my_name  =  "강윤호"
    print(  my_name  )


  </pre>
  <p>개행과 공백을 모두 있는 그대로 출력해요</p>
</body>
```
- 편리한 기능이지만, 꼭 필요한 상황이 아니라면 자주 사용하지 않는 것이 좋다.
    + 눈으로 볼 때는 편리
    + But, 시력이 낮은 사람들이 이용하는 스크린 리더 같은 소프트웨어에서는 올바르게 읽히지 않는 경우가 많음.

# 3.3 태그의 유형과 인라인 텍스트
## 태그의 유형
- 앞에서 배운 `<hx>`,`<p>`,`<blockquote>` 등으로 만들어진 요소들은 화면 상에서 하나의 블록을 형성한다.
    + 태그 안에 있는 요소가 그 길이에 상관 없이 한 줄을 통째로 차지
    + 한 줄 = 블록 레벨 요소
- 블록 레벨 요소 : 화면의 가로 길이를 통째로 차지하는 요소
- 인라인 레벨 요소 : 태그에 포함된 콘텐츠의 크기만큼만 공간을 차지

## 인라인 레벨 요소
### `<strong>` : 두꺼운 텍스트 표시하기
- 사용자의 눈에 띄어야 하는 중요한 내용을 강조해서 표시하기 위해
- 화면 상에서 두껍게 표시됨
```html
<body>
  <strong>인라인 텍스트라 한 줄을 독차지 하지 않고</strong>
  <strong>두껍게 표시되는 것을 확인해보세요</strong>
  <p>p 태그는 블록이라</p>  
  <p> 한 줄을 독차지합니다</p>
  <strong>인라인 텍스트라 한 줄을 독차지 하지 않고</strong>
  <strong>두껍게 표시되는 것을 확인해보세요</strong>
</body>
```
- 위 코드를 보면, `<strong>`은 인라인 레벨 요소이기 때문에 다음 줄에 작성해도 한 줄로 출력된다. `<p>`는 블록 레벨 요소이기 때문에 다음 줄에 작성하면 다음 줄에 출력된다.

### `<em>` : 이탤릭체 텍스트 표시하기
- 중요한 내용을 강조하기 위해 표시
- 이탤릭체로 표시됨 (기울어진 형태)
```html
<body>
  <strong>나는 두꺼운 텍스트가 돼</strong>
  <em>나는 기울어진 텍스트가 돼</em>
</body>
```
- 인라인 레벨 요소라 같은 줄에 두 문장이 출력된다.

### `<mark>` : 형광펜 효과가 있는 텍스트 표시하기
- 텍스트에 배경색을 추가해 표시하는 태그
```html
<body>
  <p>
    블록 레벨 텍스트 안에 
    <mark>인라인 레벨 텍스트를</mark>
    추가하여 표시한 결과
  </p>
</body>
```

## 웹 브라우저 개발자 도구
- 개발자 도구 여는 방법
  + F12를 누르기 or
  + [도구 더보기] - [개발자 도구]

- 개발자 도구에는 여러 개의 탭 메뉴가 있음
  + `Elements` 메뉴를 누르면, 현재 웹 브라우저 화면에 펼쳐져 있는 요소들을 만들 때 사용한 HTML 코드가 표시됨.
    + 마우스 커서를 올리면 화면에 해당 요소가 표시된다.
    + 이걸 이용해 블록 레벨 요소와 인라인 레벨 요소가 어떤 식으로 공간을 차지하고 있는지 한 눈에 볼 수 있다.


# 3.4 컨테이너와 전역 속성
## 컨테이너 태그 `<div>` , `<span>`
- 컨테이너 : 콘텐츠나 레이아웃에 아무런 영향도 주지 않고, 단지 다른 요소 여럿을 묶어 관리하기 편하게 만드는 역할을 하는 태그
- 콘텐츠 내용을 구분할 때
- 공통적인 스타일을 적용할 때
- 컨테이너 역할을 담당하는 2가지 대표적인 태그들
  + `<div>` : 블록 레벨의 컨테이너 요소를 만드는 태그
  + `<span>` : 인라인 레벨의 컨테이너를 만드는 태그

### `<div>` : 블록 레벨 컨테이너
```html
<body>
  <div>
    <p>행복하고 즐거워요~</p>
    <p>HTML 공부!</p>
  </div>
  <div>
    <p>슬프고 힘들어요</p>
    <p>적게 먹고 운동하는 다이어트!</p>
  </div>
</body>
```
- 각각의 컨테이너는 두 개의 `<p>` 태그를 감싸고 있다는 점이 같다.
- 그러나 각각이 포함하고 있는 내용의 주제가 공부와 다이어트로 다르다.
- 이렇게 컨테이너 요소는 '영역을 구분하는' 역할을 하기 위해 주로 사용된다
- `<div>` 태그는 블록 레벨 컨테이너를 만든다.

### `<span>` : 인라인 컨테이너
```html
<body>
  <p>
    예능 프로그램 즐겨 보시나요?
    저는 <span>골때리는 그녀들</span> 재미있게 보고 있습니다!
  </p>
</body>
```
- `<span>` 으로 골때리는 그녀들을 감쌌지만, 출력에는 아무런 영향이 없다.

## 전역 속성
- 전역 속성 (Global attributes)
  + 모든 HTML 태그에서 공통적으로 사용할 수 있는 속성
  + 종류 다양하고, 사용빈도에도 편차가 크기 때문에 모든 전역 속성을 알고 있을 필요는 없다.
- 사용 빈도 높은 전역 속성 3가지
  + id : 요소에 고유한 이름을 부여하는 식별자 역할
  + class : 요소를 그룹 별로 묶을 수 있는 식별자 역할
  + title : 요소에 대한 추가 정보를 기입할 수 있는 텍스트 속성. 사용자에게 툴팁을 제공

### `id` : 고유한 식별자
- 스트립트 및 스타일 적용 시 특정 요소를 식별하기 위해 지정
- 1 요소 1 `id`
```html
<body>
  <div id="study">
    <p>행복하고 즐거워요~</p>
    <p>HTML 공부!</p>
  </div>
  <div id="diet">
    <p>슬프고 힘들어요</p>
    <p>적게 먹고 운동하는 다이어트!</p>
  </div>
</body>
```
- 예제처럼 요소를 `id`로 식별한 다음, 스타일시트(CSS)나 스크립트(JavaScript)를 추가하는 식으로 작업함
- 작성 규칙
  + `id` 값은 사용자가 자유롭게 작성 가능
  + 공백 x
  + Tab x
  + 문자, 숫자, 하이픈(`-`), 언더스코어(`_`) 등을 사용

### `class` : 그룹화 식별자
- 여러 요소에 같은 이름의 식별자를 지정해 요소를 그룹화할 수 있는 식별자
- 하나의 요소가 여러 식별자를 가질 수도 있다.
- `id`와 달리, 각 요소가 고유한 값을 가지지 않아도 됨
- 스크립트 및 스타일 적용 시 특정 요소를 식별하지 위해 사용함
- 사용자가 원하는대로 값을 지정해도 됨
```html
<body>
  <div>
    <p class="text content">행복하고 즐거워요~</p>
    <p class="text content">HTML 공부!</p>
  </div>
  <div>
    <p class="text content">슬프고 힘들어요</p>
    <p class="text content">적게 먹고 운동하는 다이어트!</p>
  </div>
</body>
```

### `title` : 요소를 설명하는 추가 텍스트
- 요소와 관련된 추가 정보를 제공하는 텍스트
- 렌더링 되고 난 후 결과물에 마우스를 얹었을 때 툴팁(tooltip)이 나타남
  + 툴팁 : 커서를 얹었을 때 나타나는 도움말
