{백준 공부법}

[프로그래밍의 단계]
<3단계>
1. 기본적인 문법을 사용하여 간단한 문제를 해결할 수 있음 (필수)
- 도달하기 쉬움
- 신입생의 경우 최소 3월 안에는 달성해야한다.
- 학기초에 진행하는 교내 프로그래밍 경진대회 출전 가능.
- 1학년 때 배우는 프로그래밍 과목은 무난하게 들을 수 있음.

2. 기본적인 자료구조나 알고리즘을 사용하여 다양한 문제를 해결할 수 있음.(필수)
- 취업을 위한 단계
- 알고리즘 공부, 코딩테스트 준비하는 단계

3. 어려운 알고리즘을 공부하며, 어려운 문제를 해결할 수 있음.
- 선택의 영역
- 고급 알고리즘 공부
- 상위등급의 문제를 풀 수 있는 정도

[ 단계 도달 방법 ]
< 1단계 >
- 달성 기준 : SOLVED>AC 기준 class 2의 문제를 모두 풀었을 경우
- solved.ac 사용 방법 :
    -- 백준 계정으로 로그인.
    -- 문제 -> 새싹 탭에서 문제 풀기 (=튜토리얼 느낌)
    -- 문제 -> CLASS 탭에서 각 문제 풀기. 여기서 CLASS 2단계 달성하는 것이 목표
- 환경 세팅 :
    -- cph 확장 사용 : 테스트 케이스를 활용할 때 사용.


solved.ac 에서 문제 풀기 누르면 백준으로 자동 연결된다. 코드 제출하면 됨.
